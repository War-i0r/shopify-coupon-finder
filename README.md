# Coupon Finder Application

### How to use this app

Find a company that you'd like to check for savings on, which uses the Shop app (Shopify).

Type the name of that company EXACTLY into the light grey searchbox that is right above a button saying "Initiate Search!"

The program will then automate the Google Chrome instance in the background to search for these codes. Depending on the company and the found sites, this may take a decent amount of time. Be patient. This will be sped up in future updates, if possible.


After the program has finished searching for discount codes, they will appear color coded on the right hand side scroll area.

If you'd like to check these codes for validity, you can simply go to the store page for the company you just searched for.

Add any random item, and then proceed to checkout. Make sure the "add discount code" element of the UI is ***completely visible***, otherwise this will cause the program to hang and crash.

If you'd only like the first, highest discount code the program can find, check the box. Otherwise, click the button labelled "Check Codes".

If you see the discount code input box and the button next to it highlight in yellow, that means the program has found what it needed to. Wait for the program to finish, and it will send a popup tab with which codes worked, as well as the discounts each code gave.

NOTE: some codes may not work in conjunction with others. The program will not let you know that at this time, but that is planned in a future update.
